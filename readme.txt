チームで開発したアプリ「ダイエットの森」、個人で開発したアプリ「DC集計ツール」「selfDuelPlatform」のポートフォリオです。
ダイエットの森は、ダイエッター向けのカロリー計算ソフトです。
DC集計ツールは、遊戯王カードゲームのオンライン大会の競技者向けのデータ集計ソフトです。
selfDuelPlatformは、現実の遊戯王カードを触っているようにルールを無視してカードを並べられる一人回し用のソフトです。

特にselfDuelPlatformは、Xにて5万インプレッション、600いいね以上の反響をいただきました。
各アプリの詳細は、それぞれのreadmeおよびハッカソンペラ1を参照してください。
